Malaysia Macro Model

An open structural quarterly macroeconomic model for Malaysia. ~200 equations across 17 blocks, transparent and publicly inspectable. Built for research, policy analysis, and learning.

What It Does

Trace How Shocks Flow Through Malaysia's Economy

MaMaMo adapts the UK OBR/HM Treasury model architecture for Malaysian institutions. It models how oil prices, exchange rates, monetary policy, and global conditions transmit through consumption, investment, trade, fiscal, and household channels.

Run scenarios like "$200 oil on a US-Iran war" or "OPR hike to 4%" and see the full quarterly projection across GDP, inflation, unemployment, fiscal balance, Petronas profits, fuel subsidies, and more.

🛢️

Oil & Commodities

Brent crude pass-through to administered prices, Petronas profits, PITA revenue, and fuel subsidies. CPO production with El Niño sensitivity.

🏦

Monetary Policy

OPR transmission to BLR, deposits, MGS yields, credit growth, and household debt dynamics. Managed float exchange rate regime.

📊

Fiscal Detail

Government revenue (income tax, corporate tax, SST, Petronas dividends), expenditure (emoluments, debt service, subsidies, BSH transfers), and debt dynamics.

👷

Labour Market

Employment, wages, unemployment, participation rate, and ~2M foreign workers in plantations, construction, and manufacturing.

🌐

Trade & BOP

E&E exports driven by semiconductor cycle, tourism, commodities. Import content from IO tables. Current account, FDI, and portfolio flows.

🏠

Household Balance Sheet

Disposable income, EPF wealth, deposits, equities, housing loans, hire purchase. Debt-to-GDP at ~84%.

Key Assumptions

How the Model Works

Error-Correction Framework

Long-run equilibrium relationships with short-run adjustment dynamics. Variables converge to equilibrium over time.

Small Open Economy

Malaysia is a price-taker in commodity and capital markets. External variables (Brent, semiconductors, US rates) are exogenous.

Managed Float

REER is exogenous; NEER determined by relative prices. BNM manages the ringgit within a band.

Calibrated Coefficients

Parameters from BNM working papers, IMF reports, and DOSM IO tables — not yet fully estimated from raw data.

Current Status

Where the Model Stands Today

This is a research prototype, not a validated production forecasting system. Here's an honest assessment:

Area Status Detail
Structure Strong 17-block Malaysia-specific model architecture with ~200 equations
Documentation Strong Model file, glossary, and input specs are explicit and inspectable
Calibration Provisional Informed calibrations from published sources, not final econometric estimates
Data Pipeline In Progress OpenDOSM fetcher and input pack builder in place; auto-ingest pending
Scenario Readiness Usable Structured what-if exercises work well, especially for oil, policy, and trade shocks
Production Readiness Not Yet Too many external wedges and loosely governed inputs for official use

Command Line

Run from Terminal

Requires Node.js 18+. No dependencies to install.

# Baseline scenario
node bin/run-model.mjs
# Oil shock: Brent at $150, weaker ringgit
node bin/run-model.mjs --brent 150 --fx 4.20
# Fetch latest OpenDOSM data
node bin/fetch-opendosm.mjs
# Run tests
npm test
