Methodology Notes

This validation compares the model's baseline projections against actual quarterly data from OpenDOSM. Key points:

  • GDP growth: Year-on-year growth rates from gdp_qtr_nominal dataset
  • CPI inflation: Year-on-year changes from cpi_headline dataset (monthly, aggregated to quarterly)
  • Unemployment: Quarterly rates from lfs_qtr dataset
  • MAE (Mean Absolute Error): Average absolute forecast error in percentage points
  • RMSE (Root Mean Square Error): Penalizes larger errors more heavily
  • Model projections are baseline (no shock) scenarios run from each quarter's starting point
